LIFF Starter

It's a simple app to demonstrate how to integrate LIFF (LINE Frontend Frameworks). 

        import liff from '@line/liff'

liff
  .init({ 'Your LIFF ID' })
  .then(() => {
    // Do something with LIFF API.
  })
  .catch((error) => {
    console.log(error);
  })
